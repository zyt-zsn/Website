Play music with lyrics in EMMS

Although Emms has supported lyrics out of box(Lyrics), (require 'emms-lyrics) (emms-lyrics 1) but it only displays lyrics when there is lyric files in local directory. find.exe outputs file path encoded in utf-8, so you must told emms-source-file-directory-tree-find to use utf-8 parsing file pathes. (advice-add 'emms-source-file-directory-tree-find :around (lambda(orig &rest args) (let ( (coding-system-for-read 'utf-8) ) (apply orig args)))) Instead modify emms-source-file-directory-tree-find directly, I add advice to it to avoid repeating work when updated to the latest version of emms package Auto download lyrics and pop out lyric buffer when download is done.
